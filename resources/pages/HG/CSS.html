Selectors and Visuals
Concept or Topic Description
Selectors Choose what HTML element you'd like to alter by placing in front of brackets. Ex. h1 { color: black } Selecting * would select all elements.
Selecting Class Would select alter all HTML element in the specified class. Ex. .thisismyclass { color: black }
Selecting ID Would select the element with a specific ID. Ex. #thisismyid { color: black }

Selecting by Attributes Would select and change any elements with those specified attributes, Ex. [href] { color: black }, any element with href attribute would be black.
Chaining Selectors A way of specifying more particular elements. Ex. h1.special { color: black }, this selects h1 headings with the class special.

Descendants Would select all child elements of this class that are li elements. Ex. .main-list li { color: black }
Multiple Selectors Would select multiple different elements. Ex. h1, .button, #cool { color: black }
Font-Family To change the type face of the text on your webpage. Font must be installed or have a backup font. Multiple word fonts should be put in quotes.
Font-Size To change the size of text on your webpage. Typically measured in pixels/px.
Font Weight Controls how bold or thin text appears. Ex. Font-weight:bold/normal
Text Align Will align text to the parent. Ex. text-align:(left, center, right, justify - spaces out text to align with left and right side of the parent)
Color Property styles an element's foreground color. Ex. h1 { color: black }
Background Color Property styles an elements background color. Ex. h1 { background-color: black }
Opacity Used to measure how transparent an element is. Measured from 0 - 1
Background image Makes the background of an element an image. Must provide a url and path. Ex. background-image: url("www.example_image.com")
!important Will override any style.

Box Model
Concept or Topic Description
Width/Height The width and height of the content area can be specified in pixels. Ex. Height: 70px Width: 240px
Padding The amount of space between the content area and border. You can specify top, right, bottom, and left padding in that order in pixels. You can also specify shorthanded, ex. Padding: 5px, 10px would result in padding 5 pixels top and bottom, and 10 on left and right
Border The thickness and style of the border surrounder the content area and padding. Requires thickness (pixels or words), style/design of the border, and color. Ex. Border: 3px solid coral
Border Radius Can change the shape of the border from a box, to a rounded box, to a circle. Ex. border-radius: 50% would result in a circle assuming the element is same height and width
Margin The amount of space between the border and the outside edge of the element. Denoted in pixels. follows similar rules to padding with margin-top, bottom, left, and right. See examples in padding to see shorthand way of writing. Auto is another thing that can be used in margin, and that would instruct the browser to adjust the left and right margins until the element is centered within its containing element.
Min/Max width Ensures a miniumum or maximum width.
Min/Max height Ensures a miniumum or maximum height.
Overflow Controls what happens to content that spills or overflows outside its box. Common values are hidden (any content that overflows is hidden), scroll (a scrollbar will be added so that it can be viewed), visible (will be displayed outside hte box)..
Visibility Allowing elements to be hidden. Commons values are hidden (hides an element), visible (displays element), collapse (collapses element).
Box Sizing Controls the type of box model the browswer should use when interpreting a web page. The default value is content-box, could also be set to border-box.

Display and Positioning
Concept or Topic Description
Position Can take one of five values, static (default), relative, absolute, fixed, sticky.
Relative Position This value allows you to position an element relative to its default static position.
Offset Properties Usually done in tandem with Relative in order to say what its relative to. These properties are top, bottom, left and right are denoted in pixels or %. Writing top: 50px would move the box down 50 pixels.
Absolute Position When the position is set to absolute, all other elements on the page will ignore the element and act like it is not presnet on the page.
Fixed Position Keeps an element at the same point on the page regardless of scrolling.
Sticky Position Keeps an element in the document flow as the user scrolls but sticks to a specified position as the page is scrolled further.
Z-index Controls how far back or how far forward an element should appear on the web page when element overlap.
Display Value that dictates if it can share horitzontal space with other elements. Possible values include inline, block, and in-line block.
Inline Display Will change the display of designated elements to inline, the browser will render them on the same line.
Block Display This will fill the entire width of the page but their width can set as well.
Inline-Block Display Combination of inline and block. Blocks that can share the same line, height and width can be specified.
Float Used to move an element as far left/right as possible. Values would be left or right.
Clear Specifies how elements should behave when they bump into each other on the page. Values could include: left, right, both (neither side of the element will touch any other element), none (the element can touch either side).

Color
Concept or Topic Description
Hexadecimal A way to denote a specific color, starts with # followed by three or six characters. Ex . #A52A2A Brown
RGB A way to denote a specific color, has three values, each one can be a decimal value form 0-255. EX. rgb(143, 82, 143)
HSL or Hue/Saturation/Lightness Hue refers to an angle on the color wheel, saturation refers to the intensity or purity of the color. Lightness refers to how dark/light the colr is, 50% would be normal. Hue can be between 0-360, second and third numbers are %. Ex, hsl:(120, 60%, 70%).
Opacity/Alpha The amount of transparency in the color. The last number is a decimal between 0 and 1. 1 being opaque and 0 being transparent. Would be used by typing color: hsla(34, 100%, 50%, .1) Could also be used in RGB, ex. color: rgba(234, 45, 98, .33).

Typography
Fall Back Fonts Used when designating a font family, in case the primary font is not rendered approprately it will go to the next font. Serif = Traditional Sans Serif = modern
Font-Weight Controls how bold or then text appears, could have values such as bold, normal, lighter, bolder.
Font-Style By deafult is normal but is an option if wanting to have text apppear italicized.
Text-Transformation Allows text to be all uppercase or lowercase.
Letter Spacing Used to set horizantal spacing between individual characters, denoted in pixels or ems.
Word Spacing Used to set horizantal spacing between individual words, denoted in pixels or ems.
Line Height Sets how tall we want each line containing our text to be. Can use %, pixels, ems, numbers.
