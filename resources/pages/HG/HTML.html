HTML Structure
Tag Appearance Name Description
<!DOCTYPE html>* Doc Type Declaration Instruction informing browswer of doc type, must be the first line
<html></html>* HTML Tags Are the tag containers for all of the HTML code
<head></head> Head Tags Element that contains meta data for the page, placed between HTML and body tags
<title></title> Title Tags Always contained within the head tags. This title will not show up on the page, but on browsers and the tab
<a></a> Anchor Tags Defines a hyperlink to another page or another place on the current page. Requires href attribute specifying the destination of the link. Optional attribute "target" setting it equal to '_blank' will open a new tab.

HTML Elements
<body></body> Body Tags Contains all of an HTML document's content; headings, tables, paragraphs, etc.
<h1></h1> Headings Tags titles or subtitles that you want to display on a webpage. H1 to H6.
<div></div> Divison Tags Defines a division or section in an HTML document.
<p></p> Paragraph Tags Defines a paragraph.
<span></span> Span Tags Used to target a small piece of HTML content.
<em></em> em Tags Renders HTML text in italics
<strong></strong> Strong Tags Renders HTML text bold.
<br> Break Line Create line break in HTML text. Self closing tag.
<ul></ul> Unordered List Lists out components in a bulleted list.
<ol></ol> Ordered List Lists out components in a numbered list.
<li></li> Li Tags Contains text found in a list.
<img> Image Tag Self closing tag to display image on page. Require src and URL/local path to image. Including the 'alt' attribute will provide text in place of the image if it cannot render.
<video></video> Video Tags Used for adding an in page video, requires src link/local path, height/width, 'controls' in order to allow play/pause. Additional text between tags will display if video can not be viewed. 'Video Not Supported'

HTML Tables
<table></table> Table Tags Contains tabular data.
<tr></tr> Table Row Tags Creates a new table row.
<td></td> Table Data Tags The tag to hold cell data.
<th></th> Table Heading Tags Contains a table headings, must still be placed in table row tags. Should include attribute scope assigned to either 'col' or 'row'. Could also contain attribute colspan or rowspan to denote the space it shoud take
<tbody></tbody> Table Body Tags Container for all primary table data.
<thead></thead> Table Head Tags Normally contains table headings/rows.
<tfoot></tfoot> Table Foot Tags Contains tabular data normally towards the bottom of the table.

HTML Semantics
<header></header> Header Tags Used instead of div tags to more accurately describe the portion of HTML
<nav></nav> Navigation Tags Define a block of navigation links such as menus or table of contents
<main></main> Main Tags Used to encapsulate the dominant content within a webpage
<footer></footer> Footer Tags Typically at the bottom of the page and usually contains contact information, copyright info, etc
<section></section> Section Tags Defines elements in a document such as chapters, or areas with the same theme
<article></article> Article Tags Defines/holds content that makes sense on its own such as articles, blogs, and comments
<aside></aside> Aside Tags Used to mark additional information that can enhance another element but isnt required to understand the primary content like quotes, comments, and additional information
<figure></figure> Figure Tags Element used to encapsulate media such as image, video, etc.
<figcaption></figcaption> Figure Caption Tags Used to add a caption to a figure, placed within figure tags
<audio></audio> Audio Tags Used to embed audio content
<source> Source Tag Placed inside audio tags which references the file and type. Ex. source src='myaudio.mp3' type ='audio/mp3'
<embed> Embed Tag Can embed any media content, self closing tag. Still requires src reference within tag ex. Gif. This is 'outdated' and used less
